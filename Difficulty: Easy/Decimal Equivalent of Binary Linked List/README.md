<h2><a href="https://www.geeksforgeeks.org/problems/decimal-equivalent-of-binary-linked-list/1?page=3&category=Linked%20List&sortBy=submissions">Decimal Equivalent of Binary Linked List</a></h2><h3>Difficulty Level : Difficulty: Easy</h3><hr><div class="problems_problem_content__Xm_eO" style="--text-color: var(--problem-text-color);"><p><span style="font-size: 18.6667px;">You are given a singly linked list where each node contains either <strong>0</strong> or <strong>1</strong>. The linked list represents a binary number, where the head node is the <strong>most significant bit (MSB)</strong>.&nbsp;</span></p>
<p><span style="font-size: 18.6667px;">Convert this binary number into its <strong>decimal </strong>equivalent. </span><span style="font-size: 18.6667px;">If the linked list is empty, it represents the number 0.</span></p>
<p><span style="font-size: 18.6667px;">Since the result can be very large, return the answer modulo <strong>10<sup>9</sup>+7.</strong></span></p>
<p><span style="font-size: 14pt;"><strong>Examples:</strong></span></p>
<pre><span style="font-size: 14pt;"><strong>Input: </strong>LinkedList: 1 -&gt; 1 -&gt; 1 -&gt; 0<br> &nbsp;  <img src="https://media.geeksforgeeks.org/img-practice/prod/addEditProblem/927016/Web/Other/blobid2_1777021009.png" height="100"><br><strong>Output: </strong>14<strong><br></strong><strong>Explanation: </strong>1*2<sup>3</sup>&nbsp;+ 1*2<sup>2</sup>&nbsp;+&nbsp;1*2<sup>1</sup> + 0*2<sup>0</sup> =&nbsp; 8 + 4 + 2 + 0 = 14</span></pre>
<pre><span style="font-size: 14pt;"><strong>Input: </strong>LinkedList: 0 -&gt; 1 -&gt; 1<br>     <img src="https://media.geeksforgeeks.org/img-practice/prod/addEditProblem/927016/Web/Other/blobid1_1777014738.png" height="100"><br><strong>Output: </strong>3<br><strong>Explanation: </strong>0*2<sup>2</sup>&nbsp;+ 1*2<sup>1</sup> + 1*2<sup>0</sup> =&nbsp; 1 + 2 + 0 = 3</span></pre>
